## 选项页如何支持暗色模式

> Chrome 开启暗黑模式：地址栏输入 `chrome://flags` ，搜索关键字 `Force Dark Mode for Web Contents` ，选择 **Enabled**，重启浏览器。
>
> 由社区用户 **lock heart** 提供。
>
> 注意：这种方式小的选项页在配色上稍有些奇怪，但大体的主界面是没问题的。（建议谨慎使用）
