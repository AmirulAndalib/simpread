含义
---

> 可以将 `阅读模式` 的内容转换为 `各种文档` 的形式下载到本地，又称为 **基础导出功能**。

包含
---

- Markdown · 离线 Markdown

- PDF（使用浏览器打印）

- PNG

- HTML · [离线HTML](离线HTML)

- 复制 Markdown 到剪切板

- [临时页面](临时页面)

使用
---

>进入 `阅读模式` → 右下角控制栏 → 动作
