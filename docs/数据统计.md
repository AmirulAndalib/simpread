# 描述

> 统计稍后读的相关内容，并以 Chart 图形式展示。

# 地址

http://localhost:7026/database/statistics

# 预览

![](https://s1.ax1x.com/2023/02/23/pSxJ6bt.png)

# 统计维度

- 产生了多少稍后读

- 产生了多少条标注

- 产生最多稍后读的月份

- 共使用多少天

- 收藏最多的来源

- 标注种类统计
